# Enny's Kitchen Pro VS Code Project

This is a complete animated food ordering landing page for **Enny's Kitchen**.

## Included
- Orange and lemon color theme
- Smooth reveal animations
- Hero section
- Dynamic Today's Menu section
- Main menu cards
- Add to cart
- Cart drawer
- WhatsApp cart checkout
- Brand social links
- Brand address section
- Gallery placeholders
- Brand logo slot

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/`

## Edit first
### 1. WhatsApp number
Open `script.js` and replace:
`const WHATSAPP_NUMBER = "2348000000000";`

### 2. Brand address
Update:
`const BRAND_ADDRESS = "12 Market Road, Bodija, Ibadan, Oyo State";`

### 3. Social media links
In `index.html`, replace all `yourpage` placeholders with your real social handles.

### 4. Today's menu
In `script.js`, edit the `dynamicTodayMenu` array.

### 5. Food images
Replace the placeholder SVGs in `assets/` with your real dish photos and update paths in `index.html`.

### 6. Logo
Replace the circular `E` mark in `index.html` with your real logo.

## Run
Open the project in VS Code and launch `index.html` in your browser, or use Live Server.
